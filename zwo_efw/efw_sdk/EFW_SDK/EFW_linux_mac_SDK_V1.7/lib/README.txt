Applicable platform:
ubuntu:x86, x64
armv6: raspberry pi
armv5: armv5 Soft-Float
mac os: mac32, mac64
armv7: raspberry pi2
armv8: arm 64bit

$ sudo install efw.rules /lib/udev/rules.d
or
$ sudo install efw.rules /etc/udev/rules.d
and reconnect filter wheel, then filter wheel can be opened without root
